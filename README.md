# Prometheus Grafana Practice Project

This is a Linux monitoring practice project.

It includes:

- Prometheus
- Grafana
- Node Exporter
- Linux systemd service setup

## Project Purpose

This project is created for learning how to install and configure Prometheus and Grafana on Linux without Docker.

## Features

- Prometheus monitoring server
- Grafana dashboard UI
- Node Exporter Linux metrics
- Systemd service setup
- No Docker required

## Access URLs

Prometheus:

http://localhost:9090

Grafana:

http://localhost:3000

Node Exporter:

http://localhost:9100/metrics
